Amazonや楽天市場、Yahoo!ショッピングのアフィリエイトリンクを綺麗にかんたんにまとめて表示できるカスタムブロックを追加。ASPで取得したアフィリエイトリンクをそのまま貼り付けるだけで、綺麗なボタンのリンクが作れます。

== Description ==

Amazonや楽天市場、Yahoo!ショッピングを始めとする通販サイトの商品リンクをかんたんに作れます。
ASPで取得したリンクコードをそのまま貼り付けるだけで、整った商品リンクが完成します。
